PARTNER LOGOS
=============

Naming convention: <slug>-logo.<ext>
Where <slug> is the partner name, lowercased, accents stripped, spaces → hyphens.

Examples:
  ELEVATE      → elevate-logo.svg
  FLORATOIT    → floratoit-logo.svg
  Iko Belgium  → iko-belgium-logo.svg

Preferred format: SVG (vector, transparent background)
Fallback order tried automatically: .png → .svg → .webp → .jpg → text fallback

Dimensions: keep the original aspect ratio; the UI displays logos at max-width 180 px / height 56 px.
Background: transparent or white — the logo strip uses a light neutral (#F7F8FA) background.

To add a new partner logo:
  1. Save the file here as <slug>-logo.<ext>
  2. No code changes needed — the slug is derived automatically from the partner name.
